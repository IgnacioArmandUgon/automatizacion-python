# Curso Automatización Python

Este repositorio contiene material del curso de automatización con Python.

## Requisitos

## Python

Validar: python --version

## Como levantar

### Crear entorno virtual:

```
python -m venv venv
```

### Activarlo:

```
venv\Scripts\activate
```

### Instalar

```
pip install watchdog
pip install pyinstaller
```

### Crear ejecutable de ordenar archivo

```
pyinstaller --noconsole --onefile ordenar_archivos
```
